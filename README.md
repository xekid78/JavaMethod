# JavaMethod
メソッド

## 処理
sumGokei()メソッドを作り、その中で計算を行う。

## コード
```
public class Method {

	public static void main(String[] args) {
		sumGokei();

	}

	public static void sumGokei() {
		int sum = 5 * 8;
		System.out.println(sum);

	}

}
```
